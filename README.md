# Machine Learning for Mechanical Properties of MXenes

This repository contains machine learning models (XGBoost, SVR, RandomForest) used to predict the mechanical properties of MXenes, specifically focusing on:
- Bulk Modulus（B）
- Shear Modulus （G）
- Young's Modulus (E)
- Fracture Toughness (KIC)



## Requirements
To run the code, you need to install the following Python libraries:
